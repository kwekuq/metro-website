Flexible layouts

A flexible layout abandons the idea of the lements on a page having fixed width and height in favour of allowing them to change their dimensions to accommodate changes in the size of the browser window and font.

The way in which elements behave when either of these changes occurs depends on the units that have been used to specify their size. The table below summarizes the behaviour of lengths specified using the various units available in CSS.


Units Changing font size Changing window size
px, pt, pc, mm, cm, in No effect No effect
em, ex No effect No effect, unless the parent element's lenth is specified in em or ex
% Lengths change proportionally Lengths change proportionally, unless the parent element's lengths is specified absolutely or in em or ex

Web Graphics

The image on the screen is a rectangular array of ssmall square dots of colour, called pixels. The size of the pixels varies, usually they are between about one hundredth and one seventieth of an inch across, which means that individual pixles cannot be distinguished by the naked eye. When the pixels on the screen are set to suitable colours, the merged image produce the image we see, which to all intents and purposes is made up of continuous tones.

Some information on graphic types

Bitmapped image file formats
GIF GIF images use indexed colour, with 8-bit index values and therefor a maximum of 256 colours in the palette.
JEPG JPEG files use full 24-bit colour; they do not provide any sort of supprt for transparency.
PNG PNG format was devised to provide a replacement for GIF.
